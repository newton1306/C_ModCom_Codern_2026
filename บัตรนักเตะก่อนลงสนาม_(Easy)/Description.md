# บัตรนักเตะก่อนลงสนาม_(Easy)

ก่อนการแข่งขันฟุตบอลโลก เจ้าหน้าที่ต้องพิมพ์บัตรประจำตัวนักเตะก่อนลงสนาม

บัตรแต่ละใบประกอบด้วยชื่อนักเตะและเบอร์เสื้อ

จงเขียนโปรแกรมรับชื่อนักเตะและเบอร์เสื้อ แล้วพิมพ์บัตรนักเตะตามรูปแบบที่กำหนด

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | สตริง `name` แทนชื่อนักเตะ โดยไม่มีช่องว่าง |
| บรรทัดที่ 2 | จำนวนเต็ม `number` แทนเบอร์เสื้อ |

โดยมีเงื่อนไข

```text
1 <= ความยาวของ name <= 30
1 <= number <= 99
```

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | แสดง `Player: name` |
| บรรทัดที่ 2 | แสดง `Number: number` |
| บรรทัดที่ 3 | แสดง `Ready to play!` |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|--------------------------|
| Messi<br>10 | Player: Messi<br>Number: 10<br>Ready to play! |
| Ronaldo<br>7 | Player: Ronaldo<br>Number: 7<br>Ready to play! |
