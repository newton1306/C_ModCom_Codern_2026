# แต้มรอบแบ่งกลุ่ม_(Easy)

ในรอบแบ่งกลุ่มของฟุตบอลโลก ทีมหนึ่งจะได้ 3 แต้มเมื่อชนะ 1 นัด ได้ 1 แต้มเมื่อเสมอ 1 นัด และไม่ได้แต้มเมื่อแพ้

ให้รับจำนวนครั้งที่ชนะ เสมอ และแพ้ จากนั้นแสดงแต้มรวมของทีม

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนเต็ม `w` แทนจำนวนครั้งที่ชนะ โดยที่ 0 <= w <= 10 |
| บรรทัดที่ 2 | จำนวนเต็ม `d` แทนจำนวนครั้งที่เสมอ โดยที่ 0 <= d <= 10 |
| บรรทัดที่ 3 | จำนวนเต็ม `l` แทนจำนวนครั้งที่แพ้ โดยที่ 0 <= l <= 10 |

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | แต้มรวมของทีม |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|--------------------------|
| 2<br>1<br>0 | 7 |
| 1<br>0<br>2 | 3 |
| 0<br>3<br>0 | 3 |
