# คลังคะแนนนักศึกษาCPE40 (Hard)

นักศึกษา CPE 40 ทำแบบฝึกหัดเขียนโปรแกรมต่อเนื่อง `N` รอบ โดยได้คะแนนรวมโบนัสหรือคะแนนติดลบเป็น `A1, A2, ..., AN` อาจารย์ต้องการวิเคราะห์ช่วงการฝึกทั้งหมด `Q` ครั้งว่า หากนำคะแนนเฉพาะรอบ `L` ถึง `R` มาเรียงจากน้อยไปมาก คะแนนลำดับที่ `K` คือเท่าใด

ค่าที่ซ้ำกันต้องนับแยกกัน ตัวอย่างเช่น `2 2 7` มีค่าลำดับที่สองเป็น `2`

## ข้อมูลนำเข้า (Input)

| ลำดับ | รายละเอียด |
|---|---|
| บรรทัดที่ 1 | `N Q` (`1 <= N,Q <= 200000`) |
| บรรทัดที่ 2 | `A1 ... AN` (`-10^9 <= Ai <= 10^9`) |
| `Q` บรรทัดถัดไป | `L R K` โดย `1 <= L <= R <= N` และ `1 <= K <= R-L+1` |

## ข้อมูลส่งออก (Output)

แสดงคำตอบของแต่ละคำถามคนละบรรทัด

## ตัวอย่าง

| ข้อมูลนำเข้า | ข้อมูลส่งออก |
|---|---|
| `7 4`<br>`5 1 5 -2 9 1 3`<br>`1 7 4`<br>`2 5 2`<br>`3 3 1`<br>`4 7 3` | `3`<br>`1`<br>`5`<br>`3` |

**หมายเหตุ:** ไม่อนุญาตให้เรียงข้อมูลใหม่แยกกันสำหรับทุกคำถาม
