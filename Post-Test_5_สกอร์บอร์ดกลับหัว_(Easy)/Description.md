# Post-Test_5_สกอร์บอร์ดกลับหัว_(Easy)

ในการแข่งขันฟุตบอลโลก สกอร์บอร์ดของสนามเกิดขัดข้อง ทำให้ตัวเลขบนจอถูกมองแบบกลับหัว 180 องศา

ตัวเลขบางตัวเมื่อกลับหัวแล้วยังสามารถอ่านเป็นตัวเลขได้ แต่บางตัวจะกลายเป็นรูปที่อ่านไม่ได้

ให้รับรหัสตัวเลขบนสกอร์บอร์ดเป็นสตริง `S` แล้วหาว่าเมื่อกลับหัวทั้งป้ายแล้วจะอ่านได้เป็นเลขอะไร

ถ้ามีตัวเลขอย่างน้อยหนึ่งตัวที่กลับหัวแล้วอ่านไม่ได้ ให้แสดง `INVALID`

เมื่อตัวเลขทั้งป้ายถูกกลับหัว ลำดับของตัวเลขจะกลับด้านด้วย

## ตัวเลขที่กลับหัวได้

| **ตัวเลขเดิม** | **เมื่อกลับหัว** |
|---------------|------------------|
| `0` | `0` |
| `1` | `1` |
| `6` | `9` |
| `8` | `8` |
| `9` | `6` |

ตัวเลข `2`, `3`, `4`, `5` และ `7` ถือว่ากลับหัวแล้วอ่านไม่ได้

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | สตริง `S` แทนรหัสตัวเลขบนสกอร์บอร์ด |

โดยมีเงื่อนไข

```text
1 <= ความยาวของ S <= 100
S ประกอบด้วยตัวเลข 0 ถึง 9 เท่านั้น
S จะไม่มีเลข 0 นำหน้า ยกเว้นกรณีที่ S เป็น 0 เพียงหลักเดียว
```

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | ถ้ากลับหัวแล้วอ่านได้ ให้แสดงรหัสใหม่หลังกลับหัว หรือแสดง `INVALID` ถ้าอ่านไม่ได้ |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|--------------------------|
| 69 | 69 |
| 190 | 061 |
| 123 | INVALID |
