# Post-Test_4_รหัสเคาะประตูห้องVAR_(Easy)

ในฟุตบอลโลก ห้อง VAR มีประตูลับที่เปิดได้ด้วยรหัสเคาะประตู ผู้ตัดสินต้องเคาะตัวเลขทีละตัว โดยแต่ละตัวเป็นเลขหลักเดียวตั้งแต่ `1` ถึง `9`

เมื่อเคาะเลข `0` หมายถึงจบรหัส และเลข `0` จะไม่ถูกนำไปคิดเป็นส่วนหนึ่งของรหัส

ระบบของห้อง VAR จะตรวจรหัสด้วยกฎพิเศษดังนี้

```text
ผลรวมตัวเลขตำแหน่งคี่ - ผลรวมตัวเลขตำแหน่งคู่
```

ถ้าค่าที่ได้หารด้วย `11` ลงตัว ประตูจะเปิด

ตำแหน่งของตัวเลขเริ่มนับจาก `1` โดยเลขตัวแรกอยู่ตำแหน่งที่ `1`

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนเต็มหลายจำนวน แทนรหัสที่เคาะเข้ามาทีละตัว และจบด้วยเลข `0` |

โดยมีเงื่อนไข

```text
ตัวเลขก่อน 0 มีค่าอยู่ในช่วง 1 ถึง 9
มีเลข 0 ปิดท้ายเสมอ
จำนวนตัวเลขก่อน 0 ไม่เกิน 1000
```

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | แสดง `OPEN` ถ้าประตูเปิดได้ หรือแสดง `LOCKED` ถ้าประตูเปิดไม่ได้ |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|--------------------------|
| 1 2 1 0 | OPEN |
| 9 1 3 0 | OPEN |
| 1 2 3 0 | LOCKED |
