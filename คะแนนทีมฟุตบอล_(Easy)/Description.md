# คะแนนทีมฟุตบอล\_(Easy)

ในการแข่งขันฟุตบอลลีกหนึ่งฤดูกาล แต่ละทีมจะได้คะแนนสะสมตามผลการแข่งขัน ดังนี้

- ชนะ (Win) ได้ `3` คะแนน
- เสมอ (Draw) ได้ `1` คะแนน
- แพ้ (Lose) ได้ `0` คะแนน

ฝ่ายสถิติต้องการวัดความสม่ำเสมอของทีม จึงใช้ค่า "คะแนนเฉลี่ยต่อนัด" (Points Per Game) ซึ่งคำนวณจากคะแนนสะสมรวมหารด้วยจำนวนนัดที่ลงแข่งทั้งหมด

ให้รับจำนวนนัดที่ทีมหนึ่งชนะ เสมอ และแพ้ จากนั้นแสดงคะแนนเฉลี่ยต่อนัดของทีมนั้นเป็นทศนิยม `2` ตำแหน่ง

## ข้อมูลนำเข้า (Input)

| **ลำดับ**   | **รายละเอียด**                                                    |
| ----------- | ----------------------------------------------------------------- |
| บรรทัดที่ 1 | จำนวนเต็ม `W`, `D` และ `L` แทนจำนวนนัดที่ชนะ เสมอ และแพ้ ตามลำดับ |

โดยมีเงื่อนไข

```text
0 <= W, D, L <= 1000000
1 <= W + D + L
```

## ข้อมูลส่งออก (Output)

| **ลำดับ**   | **รายละเอียด**                               |
| ----------- | -------------------------------------------- |
| บรรทัดที่ 1 | คะแนนเฉลี่ยต่อนัดของทีม เป็นทศนิยม 2 ตำแหน่ง |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
| ------------------------ | ------------------------ |
| 10 5 3                   | 1.94                     |
| 3 0 0                    | 3.00                     |

## คำอธิบายตัวอย่าง

ตัวอย่างแรก ทีมชนะ `10` นัด เสมอ `5` นัด แพ้ `3` นัด

```text
คะแนนสะสม = 3 x 10 + 1 x 5 = 35
จำนวนนัด   = 10 + 5 + 3 = 18
เฉลี่ย      = 35 / 18 = 1.9444...
```

ปัดเป็นทศนิยม `2` ตำแหน่งได้ `1.94`

ตัวอย่างที่สอง ทีมชนะทั้ง `3` นัด ได้คะแนนเต็ม `9` จาก `3` นัด เฉลี่ยนัดละ `3.00`
