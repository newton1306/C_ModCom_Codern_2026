# ขบวนรถไฟแฟนบอลโลก (Hard)

ขบวนรถไฟพิเศษรับส่งแฟนบอลโลกมีตู้ `N` ตู้เรียงกัน แต่ละตู้มีคะแนนความสนุกของแฟนบอลหนึ่งค่า เจ้าหน้าที่ต้องประมวลผลคำสั่ง `Q` คำสั่ง

- `ADD L R X` เพิ่มคะแนนความสนุกของตู้ทุกตู้ในช่วง `[L,R]` ขึ้น `X`
- `REV L R` กลับลำดับตู้ในช่วง `[L,R]`
- `SUM L R` หาผลรวมคะแนนความสนุกของตู้ในตำแหน่ง `[L,R]` ตามลำดับปัจจุบัน

ตำแหน่งของตู้เปลี่ยนตามคำสั่ง `REV` แต่คะแนนความสนุกจะติดไปกับตู้นั้น

## ข้อมูลนำเข้า (Input)

| ลำดับ | รายละเอียด |
|---|---|
| บรรทัดที่ 1 | `N Q` (`1 <= N,Q <= 200000`) |
| บรรทัดที่ 2 | คะแนนความสนุกเริ่มต้น `A1 ... AN` (`-10^9 <= Ai <= 10^9`) |
| `Q` บรรทัดถัดไป | คำสั่งตามรูปแบบด้านบน โดย `1 <= L <= R <= N`, `-10^9 <= X <= 10^9` |

รับประกันว่าผลรวมทุกคำตอบอยู่ในช่วงจำนวนเต็ม 64 บิตแบบมีเครื่องหมาย

## ข้อมูลส่งออก (Output)

สำหรับทุกคำสั่ง `SUM` แสดงผลรวมคนละบรรทัด

## ตัวอย่าง

| ข้อมูลนำเข้า | ข้อมูลส่งออก |
|---|---|
| `6 6`<br>`1 2 3 4 5 6`<br>`SUM 1 6`<br>`REV 2 5`<br>`SUM 2 4`<br>`ADD 3 6 10`<br>`REV 1 6`<br>`SUM 1 3` | `21`<br>`12`<br>`41` |
