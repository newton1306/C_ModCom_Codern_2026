# Student_Card

จงเขียนโปรแกรมภาษา C เพื่อแสดงบัตรข้อมูลนักเรียนตามรูปแบบที่กำหนด โดยต้องแสดงชื่อและรหัสนักศึกษาให้ตรงกับตัวอย่าง

## ข้อมูลนำเข้า (Input)

ไม่มีข้อมูลนำเข้า

## ข้อมูลส่งออก (Output)

แสดงข้อมูล 2 บรรทัดดังนี้

```text
Name    : Alice
ID      : 69070503400
```

## ตัวอย่างข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|---------------------------|
| ไม่มี | Name&nbsp;&nbsp;&nbsp;&nbsp;: Alice<br>ID&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;: 69070503400 |
