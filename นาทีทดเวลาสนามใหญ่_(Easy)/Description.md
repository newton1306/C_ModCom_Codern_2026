# นาทีทดเวลาสนามใหญ่_(Easy)

การแข่งขันฟุตบอลโลกหนึ่งนัดมีเวลาปกติ 90 นาที และอาจมีทดเวลาบาดเจ็บในครึ่งแรกกับครึ่งหลัง

ให้รับจำนวนนาทีทดเวลาของครึ่งแรกและครึ่งหลัง จากนั้นแสดงเวลารวมทั้งหมดของการแข่งขัน

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนเต็ม `a` แทนนาทีทดเวลาครึ่งแรก โดยที่ 0 <= a <= 20 |
| บรรทัดที่ 2 | จำนวนเต็ม `b` แทนนาทีทดเวลาครึ่งหลัง โดยที่ 0 <= b <= 20 |

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | เวลารวมทั้งหมดของการแข่งขันเป็นนาที |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|--------------------------|
| 2<br>5 | 97 |
| 0<br>0 | 90 |
| 4<br>6 | 100 |
