# ตรวจรหัสทีมCPE40_(Medium)

ฝ่ายลงทะเบียนของ CPE 40 ต้องการตรวจรหัสทีมฟุตบอลโลก โดยรหัสที่ถูกต้องต้องขึ้นต้นด้วย `CPE40-` และตัวอักษรหลังเครื่องหมาย `-` ต้องเป็นตัวพิมพ์ใหญ่หรือตัวเลขเท่านั้น

ให้รับรหัสทีมหนึ่งรหัส แล้วตรวจว่าถูกต้องหรือไม่

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | สตริง `code` ความยาวไม่เกิน 49 ตัวอักษร และไม่มีช่องว่าง |

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | ถ้ารหัสถูกต้อง แสดง `Valid CPE 40 Code` |
| บรรทัดที่ 1 | ถ้ารหัสไม่ถูกต้อง แสดง `Invalid CPE 40 Code` |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|--------------------------|
| CPE40-THA26 | Valid CPE 40 Code |
| CPE39-THA | Invalid CPE 40 Code |
| CPE40-abc | Invalid CPE 40 Code |
