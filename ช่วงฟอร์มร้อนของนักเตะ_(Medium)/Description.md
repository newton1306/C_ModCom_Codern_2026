# ช่วงฟอร์มร้อนของนักเตะ_(Medium)

โค้ชฟุตบอลโลกกำลังวิเคราะห์ฟอร์มการยิงของนักเตะคนหนึ่งจากการซ้อมยิงทั้งหมด `N` ครั้ง

ผลของการยิงแต่ละครั้งแทนด้วยตัวเลข

- `1` หมายถึงยิงเข้า
- `0` หมายถึงยิงไม่เข้า

นักเตะจะถือว่าอยู่ในช่วง **ฟอร์มร้อน** ถ้ามีการยิงเข้าติดต่อกันอย่างน้อย `K` ครั้ง

จงตรวจสอบว่านักเตะคนนี้มีช่วงฟอร์มร้อนหรือไม่

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนเต็ม `N` และ `K` แทนจำนวนครั้งที่ซ้อมยิง และจำนวนครั้งติดกันขั้นต่ำที่ถือว่าฟอร์มร้อน |
| บรรทัดที่ 2 | จำนวนเต็ม `N` จำนวน แทนผลการยิงแต่ละครั้ง โดยแต่ละค่าเป็น `0` หรือ `1` |

โดยมีเงื่อนไข

```text
1 <= K <= N <= 100000
```

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | แสดง `HOT` ถ้ามีการยิงเข้าติดต่อกันอย่างน้อย `K` ครั้ง หรือแสดง `NORMAL` ถ้าไม่มี |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|--------------------------|
| 6 3<br>1 1 1 0 1 0 | HOT |
| 7 3<br>1 1 0 1 1 0 1 | NORMAL |
| 5 2<br>0 1 1 0 0 | HOT |

## คำอธิบายตัวอย่าง

ตัวอย่างที่ 1 มีการยิงเข้า 3 ครั้งติดกันตั้งแต่ครั้งที่ 1 ถึง 3 จึงแสดง `HOT`

ตัวอย่างที่ 2 มีการยิงเข้าติดกันมากที่สุดเพียง 2 ครั้ง ยังไม่ถึง `K = 3` จึงแสดง `NORMAL`
