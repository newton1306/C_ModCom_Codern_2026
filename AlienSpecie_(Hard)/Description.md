# AlienSpecie_(Hard)

สถานีอวกาศแห่งหนึ่งพบสิ่งมีชีวิตต่างดาวทั้งหมด `N` ตัว ยืนเรียงกันอยู่ในทางเดินตรวจสอบ โดยแต่ละตัวมีหมายเลขตำแหน่งตั้งแต่ `1` ถึง `N`

สิ่งมีชีวิตต่างดาวเหล่านี้ถูกแบ่งออกเป็น `K` สปีชีส์ แต่ละสปีชีส์แทนด้วยหมายเลขตั้งแต่ `1` ถึง `K`

นักวิจัยต้องการตรวจสอบจำนวนสิ่งมีชีวิตต่างดาวแต่ละสปีชีส์ในช่วงต่าง ๆ ของทางเดิน จึงมีคำถามทั้งหมด `Q` คำถาม แต่ละคำถามกำหนดช่วงตำแหน่งตั้งแต่ `L` ถึง `R`

สำหรับแต่ละคำถาม จงหาว่าในช่วงตำแหน่ง `L` ถึง `R` มีสิ่งมีชีวิตต่างดาวสปีชีส์ที่ `1`, `2`, ..., `K` อย่างละกี่ตัว

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนเต็ม `N`, `K` และ `Q` แทนจำนวนสิ่งมีชีวิตต่างดาว จำนวนสปีชีส์ และจำนวนคำถาม |
| บรรทัดที่ 2 | จำนวนเต็ม `A1 A2 ... AN` โดย `Ai` คือหมายเลขสปีชีส์ของสิ่งมีชีวิตต่างดาวที่ตำแหน่ง `i` |
| บรรทัดที่ 3 ถึง Q + 2 | จำนวนเต็ม `L` และ `R` แทนช่วงตำแหน่งที่ต้องการตรวจสอบ |

โดยมีเงื่อนไข

```text
1 <= N <= 100000
1 <= K <= 20
1 <= Q <= 100000
1 <= Ai <= K
1 <= L <= R <= N
```

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 ถึง Q | สำหรับแต่ละคำถาม ให้แสดงจำนวนเต็ม `K` จำนวน โดยจำนวนที่ `j` คือจำนวนสิ่งมีชีวิตต่างดาวสปีชีส์ที่ `j` ในช่วง `L` ถึง `R` |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|--------------------------|
| 8 4 3<br>2 1 4 1 3 2 4 1<br>1 8<br>3 6<br>5 5 | 3 2 1 2<br>1 1 1 1<br>0 0 1 0 |

## คำอธิบายตัวอย่าง

สิ่งมีชีวิตต่างดาวทั้งหมดเรียงตามตำแหน่งดังนี้

```text
ตำแหน่ง:  1 2 3 4 5 6 7 8
สปีชีส์: 2 1 4 1 3 2 4 1
```

คำถามแรกคือช่วง `1` ถึง `8`

```text
สปีชีส์ 1 มี 3 ตัว
สปีชีส์ 2 มี 2 ตัว
สปีชีส์ 3 มี 1 ตัว
สปีชีส์ 4 มี 2 ตัว
```

จึงได้ผลลัพธ์

```text
3 2 1 2
```

คำถามที่สองคือช่วง `3` ถึง `6`

สิ่งมีชีวิตต่างดาวในช่วงดังกล่าวมีสปีชีส์

```text
4 1 3 2
```

แต่ละสปีชีส์มีอย่างละ 1 ตัว จึงได้ผลลัพธ์

```text
1 1 1 1
```

คำถามที่สามคือช่วง `5` ถึง `5`

ในช่วงนี้มีสิ่งมีชีวิตต่างดาวเพียงตัวเดียว ซึ่งเป็นสปีชีส์ 3 จึงได้ผลลัพธ์

```text
0 0 1 0
```
