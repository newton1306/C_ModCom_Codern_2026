# ถนนเจ้าภาพฟุตบอลโลก (Hard)

ประเทศเจ้าภาพฟุตบอลโลกมีเมืองจัดการแข่งขันและเมืองแฟนโซนรวม `N` เมือง พร้อมถนนสองทิศทางที่เสนอสร้าง `M` เส้น ถนนแต่ละเส้นมีค่าใช้จ่าย ฝ่ายจัดการแข่งขันต้องการทราบ

1. ค่าใช้จ่ายต่ำที่สุดที่ทำให้แฟนบอลเดินทางถึงกันได้ทุกเมือง ซึ่งคือต้นไม้แผ่ทั่วขั้นต่ำ (Minimum Spanning Tree)
2. ค่าใช้จ่ายต่ำที่สุดของแผนถนนสำรองที่ใช้ชุดถนนไม่เหมือนแผนในข้อ 1 อย่างน้อยหนึ่งเส้น ค่าใช้จ่ายนี้อาจเท่ากับข้อ 1

หากสร้างต้นไม้แผ่ทั่วไม่ได้ ให้แสดง `-1 -1` หากสร้างได้เพียงชุดเดียว ให้ค่าที่สองเป็น `-1`

ถ้ามี MST หลายชุด สามารถเลือก MST ชุดใดก็ได้เป็นชุดแรก แล้วค่าที่สองต้องเป็นค่าต่ำที่สุดของต้นไม้ที่มีชุดถนนต่างจากชุดที่เลือกนั้น

## ข้อมูลนำเข้า (Input)

| ลำดับ | รายละเอียด |
|---|---|
| บรรทัดที่ 1 | `N M` (`2 <= N <= 200000`, `1 <= M <= 300000`) |
| `M` บรรทัดถัดไป | `u v w` (`1 <= u < v <= N`, `0 <= w <= 10^9`) ถนนแต่ละเส้นถือเป็นคนละเส้นแม้ปลายซ้ำกัน |

## ข้อมูลส่งออก (Output)

แสดงจำนวนเต็มสองจำนวน คือค่าน้ำหนักของ MST และต้นไม้สำรองตามลำดับ

## ตัวอย่าง

| ข้อมูลนำเข้า | ข้อมูลส่งออก |
|---|---|
| `3 3`<br>`1 2 1`<br>`2 3 2`<br>`1 3 3` | `3 4` |
