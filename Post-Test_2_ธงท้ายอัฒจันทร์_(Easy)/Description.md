# Post-Test_2_ธงท้ายอัฒจันทร์_(Easy)

แฟนบอลในสนามฟุตบอลโลกกำลังทำธงเชียร์เป็นรูปสามเหลี่ยมบนอัฒจันทร์

ธงเชียร์มีทั้งหมด `N` แถว โดยแถวที่ `i` จะมีสัญลักษณ์ทั้งหมด `i` ตัว

แถวทั่วไปให้ใช้สัญลักษณ์ `*` แต่แถวสุดท้ายเป็นฐานของธง จึงให้ใช้สัญลักษณ์ `#` แทน

สามารถเลือกทิศทางการวาดได้ 2 แบบ

- `mode = 1` วาดธงแบบปกติ โดยชิดซ้าย
- `mode = -1` วาดธงแบบกลับซ้ายขวา โดยชิดขวา

จงเขียนโปรแกรมเพื่อพิมพ์ธงเชียร์ตามขนาดและทิศทางที่กำหนด

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนเต็ม `N` แทนจำนวนแถวของธงเชียร์ |
| บรรทัดที่ 2 | จำนวนเต็ม `mode` แทนทิศทางการวาด โดย `1` คือชิดซ้าย และ `-1` คือชิดขวา |

โดยมีเงื่อนไข

```text
1 <= N <= 50
mode เป็น 1 หรือ -1 เท่านั้น
```

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 ถึง N | แสดงธงเชียร์ตามขนาดและทิศทางที่กำหนด |

## ข้อมูลนำเข้าและส่งออก

ตัวอย่างที่ 1

**ข้อมูลนำเข้า**

```text

1
```

**ข้อมูลส่งออก**

```text
*
**
***
****
#####
```

**ข้อมูลนำเข้า**

```text

-1
```

**ข้อมูลส่งออก**

```text
    *
   **
  ***
 ****
#####
```

ตัวอย่างที่ 3

**ข้อมูลนำเข้า**

```text
1
-1
```

**ข้อมูลส่งออก**

```text
#
```
