# Square_Function

จงเขียนโปรแกรมรับจำนวนเต็ม `N` แล้วเขียนฟังก์ชันสำหรับคำนวณค่า `N` ยกกำลังสอง

## ข้อมูลนำเข้า (Input)

บรรทัดแรกรับจำนวนเต็ม `N`

## ข้อมูลส่งออก (Output)

แสดงค่าของ `N * N`

## ตัวอย่างข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|---------------------------|
| 3 | 9 |
