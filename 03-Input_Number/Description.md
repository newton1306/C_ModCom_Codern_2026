# Input_Number

จงเขียนโปรแกรมรับจำนวน 2 ค่า ได้แก่ จำนวนเต็ม `A` และจำนวนทศนิยม `B` จากนั้นแสดงค่าทั้งสองออกมาตามรูปแบบที่กำหนด

## ข้อมูลนำเข้า (Input)

บรรทัดแรก รับจำนวนเต็ม `A`

บรรทัดที่สอง รับจำนวนทศนิยม `B`

## ข้อมูลส่งออก (Output)

แสดงข้อความในรูปแบบ

```text
A = <A>, B = <B>
```

โดยแสดง `B` เป็นทศนิยม 2 ตำแหน่ง

## ตัวอย่างข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|---------------------------|
| 10<br>20.5 | A = 10, B = 20.50 |
