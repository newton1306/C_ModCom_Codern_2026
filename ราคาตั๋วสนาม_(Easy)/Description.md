# ราคาตั๋วสนาม_(Easy)

สนามฟุตบอลแห่งหนึ่งแบ่งที่นั่งออกเป็น `3` โซน คือ โซนหลังประตู โซนข้างสนาม และโซนวีไอพี โดยแต่ละโซนมีราคาตั๋วต่อใบและจำนวนตั๋วที่ขายได้แตกต่างกัน

ให้รับราคาตั๋วต่อใบและจำนวนตั๋วที่ขายได้ของทั้ง `3` โซน จากนั้นแสดงรายได้รวมจากการขายตั๋วทั้งหมด

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 ถึง 3 | จำนวนเต็ม `price` และ `qty` แทนราคาตั๋วต่อใบและจำนวนตั๋วที่ขายได้ของแต่ละโซน |

โดยมีเงื่อนไข

```text
0 <= price <= 10000
0 <= qty <= 100000
```

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | รายได้รวมจากการขายตั๋วทั้งหมด |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|--------------------------|
| 200 5000<br>500 3000<br>1500 200 | 2800000 |
| 100 0<br>0 50<br>300 10 | 3000 |

## คำอธิบายตัวอย่าง

ตัวอย่างแรก

```text
โซน 1: 200 x 5000 = 1000000
โซน 2: 500 x 3000 = 1500000
โซน 3: 1500 x 200 = 300000
```

รวมรายได้ทั้งหมด `1000000 + 1500000 + 300000 = 2800000`
