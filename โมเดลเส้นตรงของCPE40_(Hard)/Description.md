# โมเดลเส้นตรงของCPE40 (Hard)

นักศึกษา CPE 40 กำลังทดลองระบบเลือกโมเดลเชิงเส้นสำหรับคาดการณ์เวลาทำงานของโปรแกรม แต่ละโมเดลเขียนได้เป็น `y = m*x + b` ระบบเริ่มต้นโดยยังไม่มีโมเดล และรับคำสั่ง `Q` คำสั่ง

- `ADD m b` เพิ่มโมเดล `y = m*x + b` เข้าระบบ โมเดลที่เพิ่มแล้วจะไม่ถูกลบ
- `QUERY x` หาค่าเวลาคาดการณ์ `y` ที่น้อยที่สุดจากทุกโมเดล ณ ขนาดข้อมูล `x` หากยังไม่มีโมเดลให้แสดง `EMPTY`

## ข้อมูลนำเข้า (Input)

| ลำดับ | รายละเอียด |
|---|---|
| บรรทัดที่ 1 | `Q` (`1 <= Q <= 200000`) |
| `Q` บรรทัดถัดไป | `ADD m b` หรือ `QUERY x` โดย `-10^9 <= m,b,x <= 10^9` |

รับประกันว่าคำตอบของทุก `QUERY` ที่ไม่ว่างอยู่ในช่วงจำนวนเต็ม 64 บิตแบบมีเครื่องหมาย

## ข้อมูลส่งออก (Output)

แสดงค่าต่ำสุดหรือ `EMPTY` สำหรับทุกคำสั่ง `QUERY` คนละบรรทัด

## ตัวอย่าง

| ข้อมูลนำเข้า | ข้อมูลส่งออก |
|---|---|
| `7`<br>`QUERY 2`<br>`ADD 2 3`<br>`ADD -1 10`<br>`QUERY 0`<br>`QUERY 4`<br>`ADD 0 -5`<br>`QUERY 100` | `EMPTY`<br>`3`<br>`6`<br>`-90` |
