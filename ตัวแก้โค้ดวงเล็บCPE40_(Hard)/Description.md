# ตัวแก้โค้ดวงเล็บCPE40 (Hard)

นักศึกษา CPE 40 กำลังสร้างเครื่องมือตรวจ syntax สำหรับโค้ดภาษา C เครื่องมือเก็บสายอักขระวงเล็บ `S` ยาว `N` และต้องรองรับคำสั่งแก้ไขหรือตรวจสอบ `Q` คำสั่ง

- `F L R` โปรแกรมแก้โค้ดกลับขั้ววงเล็บทุกตัวในช่วง `[L,R]` โดย `(` กลายเป็น `)` และ `)` กลายเป็น `(`
- `Q L R` ตรวจว่าโค้ดส่วน `S[L..R]` มีลำดับวงเล็บที่ถูกต้องหรือไม่

ลำดับวงเล็บที่ถูกต้องต้องจับคู่ได้ครบ และเมื่ออ่านจากซ้ายไปขวาจะต้องไม่มีวงเล็บปิดที่ยังไม่มีวงเล็บเปิดให้จับคู่

## ข้อมูลนำเข้า (Input)

| ลำดับ | รายละเอียด |
|---|---|
| บรรทัดที่ 1 | `N Q` (`1 <= N,Q <= 200000`) |
| บรรทัดที่ 2 | สายอักขระ `S` ซึ่งมีเพียง `(` และ `)` |
| `Q` บรรทัดถัดไป | คำสั่ง `F L R` หรือ `Q L R` (`1 <= L <= R <= N`) |

## ข้อมูลส่งออก (Output)

สำหรับคำสั่ง `Q` แสดง `YES` หรือ `NO` คนละบรรทัด

## ตัวอย่าง

| ข้อมูลนำเข้า | ข้อมูลส่งออก |
|---|---|
| `8 6`<br>`(()())()`<br>`Q 1 8`<br>`Q 2 5`<br>`F 2 3`<br>`Q 1 4`<br>`F 2 3`<br>`Q 3 6` | `YES`<br>`YES`<br>`NO`<br>`NO` |
